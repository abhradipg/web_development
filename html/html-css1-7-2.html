HTML Tags (6-19)

Question: 6

What are the little bits of text that surround elements in an html file?


Explanation:
Open and close Tags, denoted by < and > , are used to mark the beginning and end of a given element.

Question: 7

Can some elements contain other elements?



Explanation:
Many elements contain other elements, e.g. the body can contain one or more p elements, which can contain em elements and so on.

Question: 8

<!-- If this line were in your code, would you see it on your Web page? -->



Explanation:
' <!-- ' and ' --> ' are a special kind of tag called "comments", generally used by humans for humans, but they're ignored by the computer, so they don't show up on your rendered webpage

Question: 9

True or False: elements are the same as tags.



Explanation:
Elements make up an html document, while
Tags are the bits of text used to denote Elements.

Question: 10

If you want to indicate that a large passage of text is an exact quotation, what tag would you use?


Explanation:
Blockquote is for a large block of text that is being quoted verbatim from another work.

Question: 11

True or False: attributes are used to add extra info to a tag, like a unique identifier.



Explanation:
The 'id' attribute is designed to hold a unique identifier

Question: 12

If you want to have the largest heading possible, which kind of tag you would use?





Explanation:
"h1" represents the outermost heading, and is generally made the largest. The head element is for the head of the document, not for headings in the text. There is no h element.

Question: 13

What is the first character after a "<" in a closing tag?





Explanation:
For element types that require closing tags, all of them are indicated with a / followed by the type of element it is, e.g. for a p element, the close tag would be < /p >

Question: 14

What doctype is used for HTML5 documents?





Explanation:
One of the enhancements made to the HTML5 standard was to simplify the doctype declaration to be just 'html'.

Question: 15

What is the last tag in a well-behaved html file?





Explanation:
The html element is the outermost element in a file, so it must be the last closing tag.

Question: 16

What tag would you use to emphasize some text?


Explanation:
The em tag indicates text that should be given more emphasis.

Question: 17

If I told you that an open tag looked like this: <div id='mydiv' class='myclass'>, what would the closing tag look like?


Explanation:
Close tags are just like open tags, but with a prepended '/', and there are no attributes on close tags.

Question: 18

The <p> tag is used to denote:


Explanation:
P, for paragraph, is used to mark the beginning (open tag) and end (close tag) of a paragraph.

Question: 19

A ul element wouldn't be terribly useful unless it contained one or more of these elements:


Explanation:
A list, unordered or otherwise, wouldn't be very interesting if it had no List Items.
