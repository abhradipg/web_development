Explanation:
Though a browser may accept your code without any difficulty, experienced programmers have found that Best practices are a good idea.

Question: 22

Let's consider a space, or a tab or a carriage return. What are each of them referred to?


Explanation:
White space includes spaces, tabs or carriage returns, providing either horizontal or vertical separation.

Question: 23

True or False: You can use either ' (single quote) or " (double quote) to denote a string.



Explanation:
Either type of quote is acceptable, but they must match in a given string. A string that starts with " must end with ". Any 's will be considered part of the string.

Question: 24

True or False: HTML tags are case sensitive.



Explanation:
HTML tags are not case sensitive, but be wary as other things in an HTML file are

Question: 25

True or False: you do need a connection to the Internet to test your HTML code.



Explanation:
You can test your code in a browser on your local machine, either by opening your file in the browser or setting up a simple Web server running on your machine.

Question: 26

True or False: Any complete HTML5 file must contain bits of CSS and Javascript.



Explanation:
Though a modern Web page typically uses all three languages, you can create a complete correct Web page using only HTML.

Question: 27

Which of these HTML fragments would render differently from the others?





Explanation:
White space doesn't matter (multiple spaces are treated the same as a single space).
Capitalization of tag names doesn't matter (not case sensitive).
Capitalization of content does matter (text is case sensitive).

Question: 29

If you have a mistake in your HTML code, the browser will:
(only 1 correct answer!)





Explanation:
It's highly unlikely that a modern browser would crash due to mistaken HTML code.
