HTML5 and the Web (1-5)

Question: 1

What does the 'M' in HTML5 stand for?


Explanation:
HyperText Markup Language

Question: 2

How many elements should your document have?





Explanation:
Though there may be a minimum number of elements required (e.g. an html element, a head and a body) there's no limit, it just depends on how you organize your info.

Question: 3

What is CSS?
(select all that apply - 2 answers are possible!)





Explanation:
CSS is a separate language from HTML, used to control the look (style or visual theme) of a web page.

Question: 4

True or False: The main reason the Internet was created was to support the World Wide Web



Explanation:
The internet existed for many years before the World Wide Web was conceived.
